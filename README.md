酷爽英语语法
===========

### 简介

一款英语初级学习APP，并带有测试功能，还支持词库下载。云端词库由我自己写成，服务器架在SAE上，使用PHP写成，传输格式为xml.

### 依赖

android-support-v7-appcompat.jar  
android-support-v4-appcompat.jar  
open source architecture PagerSlidingTabStrip
